# List (`<ul>`, `<ol>`, `<li>`)

## Unordered (Bullet) List

```html
<ul>
  <li>Home</li>
  <li>About</li>
</ul>
```

## Ordered (Number) List

```html
<ol>
  <li>First</li>
  <li>Second</li>
</ol>
```

### Start at Specific Number

```html
<ol start="1999">
```

### Reversed

```html
<ol start="2002" reversed>
  ```

### Type

```html
<ol type="[1AaIi]">
```
