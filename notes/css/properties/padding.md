# `padding`

Area immediately around content. See [box model](../box_model.md).

```css
/* all 4 sides */
padding: 1em;

/* top and bottom | left and right */
padding: 5% 10%;

/* top | left and right | bottom */
padding: 1em 2em 2em;

/* top | right | bottom | left */
padding: 5px 1em 0 2em;

/* global values */
padding: inherit;
padding: initial;
```
