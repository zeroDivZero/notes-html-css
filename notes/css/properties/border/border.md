# `border`

Shorthand to set `border-width`, `border-style`, and `border-color`. Needs at least style to appear.

```css
/* style */
border: solid;

/* width | style */
border: 2px dotted;

/* style | color */
border: outset #f33;

/* width | style | color */
border: medium dashed green;

/* global values */
border: inherit;
border: initial;
```
