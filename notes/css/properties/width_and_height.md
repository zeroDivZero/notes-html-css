# `width`, `height`

Dimensions of element (always rectangular).

## Pixels

```css
width: 220px;
height: 150px;
```

## Font-Size Values

```css
width: 20em;
height: 10em;
```

## Percentage of Parent

```css
width: 100%;
height: 75%;
```

## Keywords

```css
width: max-content;
width: min-content;
width: auto;
```

## Global Values

```css
width: inherit;
width: initial;
```

## Min, Max

Actual value clamped to min-max range if specified:

* `min-width`, `max-width`
* `min-height`, `max-height`
